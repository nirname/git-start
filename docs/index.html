GIT

Configuring

Первоначальная настройка Git

git config --global user.name "Inanov Ivan"
git config --global user.email ivanov@example.com

.gitconfig

[alias]
  l = log --pretty=format:\"%C(yellow)%h%C(reset) | %s %C(red)[%aN]%C(green)%d%C(reset)\" --decorate --graph --all
  co = checkout
  ci = commit
  st = status
  br = branch
  mr = merge

Внешние утилиты

git config --global diff.tool meld
git config --global difftool.prompt false
git config --global core.editor /usr/bin/vim

External diff

[diff]
  tool = meld
[difftool]
  path = meld
  prompt = false

External editor

[core]
  editor = /usr/bin/vim

Init

Инициализировать новый проект c Git

git init new-project

Добавить существующий проект в Git

cd existing-project
git init .

Staging

Doing chagnes

git add .
git reset -- dont-commit.txt
git commit -m 'Here goes message'

Branches

Список веток

$ git branch --all
* master

Новая ветка

git branch feature
git checkout feature
git checkout -b feature # сокращённо

Разработка

touch D; git add .; git commit -m 'D'
touch F; git add .; git commit -m 'F'

Слияние fast-forward, "Перемотка"

git checkout master
git merge feature

Слияние --no-ff

git checkout master
git merge --no-ff feature

Undoing changes

Unstage changes

git reset --soft

Save staged changes

git reset
